# ShortyRCD

ShortyRCD is a distributed raid cooldown tracker for World of Warcraft (Midnight expansion).

## Design Philosophy

- Each player broadcasts their own cooldown usage.
- No reliance on combat log parsing of other players.
- No localization issues (spellID-based).
- Lightweight and compliant with Midnight API changes.
